#  [星宇·航图 | XY's Blog](https://xyos2.xyz/)


## 项目介绍

这是一个个人博客网站，展示我的作品和常用工具，采用现代化的浅色科技感设计。

## 主要功能

- ✨ 响应式设计，适配各种设备
- 📱 移动端友好的导航菜单
- 🎯 作品展示卡片，支持点击跳转
- 🛠️ 常用工具推荐
- 📌 实时时间显示
- 🔄 平滑滚动效果

## 技术栈

- HTML5
- CSS3
- JavaScript
- Font Awesome 图标
- Google Fonts

## 目录结构

```
.
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # JavaScript文件
├── README.md           # 项目说明文档
└── pages/              # 工具页面目录
    ├── page01.html     # 代码转换器
    ├── page02.html     # 配色生成
    ├── page03.html     # 图片优化
```

## 使用说明

1. 直接在浏览器中打开 `index.html` 文件
2. 点击导航栏切换不同区域
3. 点击作品卡片查看详情
4. 点击常用工具卡片进入对应工具页面
